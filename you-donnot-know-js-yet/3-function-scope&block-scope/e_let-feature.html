1.let进行的声明不会在其所在块作用域中进行提升

        
          if(true){
            console.log(bar);   //ReferenceError: Cannot access 'bar' before initialization
            let bar = "hello";                                    
          }
or
        
          function foo(){
            console.log(bar);  //ReferenceError: Cannot access 'bar' before initialization
            let bar = "hello";   
          }
          foo();

以后会详细讨论提升,提升是指声明会被视为存在于其所出现的作用域的整个范围内。
