函数会首先被提升

函数声明和变量声明都会被提升,但是函数会首先被提升,然后才是变量

考虑以下代码： 
        
          foo();       // 2

          var foo;

          function foo(){
            console.log(1);
          }

          function foo(){
            console.log(2);
          }
这个代码片段会被引擎理解为如下形式： 
        
          function foo(){
            console.log(2);
          }

          foo();       // 2

var foo 尽管出现在function foo()... 的声明之前,因为函数声明会被提升到普通变量之前,并且有相同的名字foo,所有被视作重复的声明（因此被忽略）,
同时在后面的foo函数声明还可以覆盖前面的foo函数声明,因此输出2

※1.javascript中如果在同一个作用域出现相同名字的声明(不论函数还是变量),后面出现的声明都将被忽略

  2.如果在嵌套的作用域中,子作用域中出现与父作用域相同名字的声明,则会覆盖父作用域中的声明
